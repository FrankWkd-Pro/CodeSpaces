# Day10:区间、树形DP
## 区间DP
- 区间类型动态规划是线性动态规划的拓展，它在分阶段划分问题时，与阶段中元素出现的顺序和由前一阶段的哪些元素合并而来有很大的关系。（例：$f[i][j]=f[i][k-1]+f[k][j]$ 
- 区间类动态规划的特点：
  - 合并：即将两个或多个部分进行整合，
  - 特征：能将问题分解成为两两合井的形式
  - 求解：对整个问题设最优值，枚举合并点，将问题分解成为左右两个部分，最后将左右两个部分的最优值进行合并得到原问颗的最优值
